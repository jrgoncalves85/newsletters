Newsletter 25: CONECTORES E PLUGS

A maioria das pessoas se preocupam apenas com que seu computador funcione, pouco se importando em saber a forma como ele faz isso. Mas, às vez é preciso desconectar alguns fios para uma eventual limpeza e na hora de conectar novamente surge a dúvida de onde ligar aquele emaranhado de fios. E, muitas vezes, é preciso chamar um técnico para "consertar" só por que a pessoa inverteu o lugar de conecatr o cabo do teclado ou da caixa de som.

A Newsletter de hoje tem como objetivo auxiliar essas pessoas a resolverem sozinhas esse tipo de problema que ela mesma pode resolver.

Importante: Na hora de conectar qualquer cabo é preciso muito cuidado, pois se forçado de forma errada é possível quebrar dos pinos de encaixe e ocasionar danos no conector ou ao aparelho.

Segue abaixo ilustrações que mostram parcialmente a parte traseira de um computador.

Saída de energia elétrica: utilizada muitas vezes para ligar o monitor, funciona como uma tomada comum.

Entrada de energia elétrica: utilizada para ligar o computador à rede elétrica (tomada)

Conector PS/2 roxo: usado para conectar teclados;

Conector PS/2 verde: usado para conectar mouse;

Porta paralela: usada para conectar impressoras;

Porta VGA: usada para conectar monitores;

Portas USB: usadas para conectar pen-drives, impressoras, etc;

Porta de rede: usada para conectar cabos de rede RJ-45 de internet;

Porta P2 Verde: usada para conectar caixas de som;

Porta P2: Rosa: usada para conectar microfones.
